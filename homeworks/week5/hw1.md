## 前四週心得與解題心得


在這個複習週，整理了過去四週的筆記，雖然也沒有照自己原先訂定的計畫，全部整理完。

從第零週的心態調整開始，也是在看那些文章時，自己才決定要整理筆記的，但 6 月份計畫開始時看那些文章並沒有覺得要整理筆記的必要， 就在這個複習週有了想留下一點自己學習過的痕跡，所以就動手開始整理了。

一開始解題時，是拿著白紙先在紙上寫下來，後來慢慢變成在 sublime 上寫 pseudo code，但自己還是無法在複習週練習一些之前認為比較有難度的題目時，可以解得更快，好像還是會卡蠻久的。

了解並且會使用 Git 也是前四週的一個很大收穫～

第四週 API 真的卡超久， 或許真的因為他是很抽象的東西，要具體化已經是一個難度，要了解也是需要時間消化，但老師教的方式有很盡量要用平易近人的方式讓大家了解了。很喜歡複習週裡有一個 Lidemy HTTP Challenge 的小遊戲，寓教於樂的目的有感受到。

第五週的 show time 看完後，真的會讓人驚訝： 原來這都是前幾週學的東西可以做出來的事！ 自信心有上升 （雖然不是自己做出來的）尤其看得懂為什麼要那樣寫後，好像比起兩個月前的自己真的學到不少，我成長了～

自己在第四週跟第五週的時間拉很長，一方面是因為家人健康狀況，要南北來回奔波，另一方面是撞牆期。現在要邁入第六週了，看到有些同學分享的心得是 ：第六週會一掃前面卡關的陰霾！好期待～

最後感謝自己有報名這個計畫～ 而且到現在還堅持著！
